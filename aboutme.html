About Me

Hi, saya adalah seorang pelajar yang berdomisisli di Surakarta. Saat ini sedang belajar pemrograman web dengan menggunakan HTML.

Saya memang masih baru dalam web development, karena itu saya tidak akan pernah berhenti belajar karena saya sangat tertarik untuk mempelajari hal-hal baru, terutama dalam hobi saya yaitu membaca dan menulis sesuatu, seperti menuliskan coding.

Saya ingin menguasai bahasa Java, HTML, CSS, dan Javascript, Phyton.
